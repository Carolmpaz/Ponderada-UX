# La Casita - Bad Bunny (D3.js Animation)

Este projeto é uma representação artística e interativa da famosa **"Casita"** do álbum *Un Verano Sin Ti*, do artista **Bad Bunny**, inspirada na cenografia vibrante utilizada em suas turnês mundiais.

O projeto foi desenvolvido inteiramente utilizando a biblioteca **D3.js** para manipulação de SVG, transformando coordenadas estáticas em uma experiência visual dinâmica, imersiva e responsiva.

---

## Funcionalidades

* **Renderização SVG de Alta Precisão**: Cada detalhe da estrutura, desde os pilares de sustentação até os gradientes das rodas, foi reconstruído elemento por elemento via código JavaScript.
* **Show de Luzes Interativo**: Ao passar o mouse sobre o desenho, um jogo de luzes de show é ativado ao fundo da casa.
* **Animação de Feixes**: Os feixes de luz possuem movimentos senoidais e cores aleatórias (Cyan, Magenta, Lime), simulando a iluminação real de um palco.
* **Filtros de Glow**: Aplicação de desfoque gaussiano nos feixes de luz para criar um efeito de iluminação neon realista.

---

## Tecnologias Utilizadas

* **JavaScript**
* **D3.js (v7)**
* **SVG (Scalable Vector Graphics)**
